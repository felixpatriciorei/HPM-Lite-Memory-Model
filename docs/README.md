# Documentation map

This folder is organized by purpose.

## Research

`docs/research/` contains result writeups, statistical methods, long-context summaries, and legacy result notes.

## Figures

`docs/figures/` contains figure design notes, figure-generation notes, figure audit material, and the legacy graph retirement manifest.

## Engineering

`docs/engineering/` contains implementation notes, logging schemas, HPM-v2 integration notes, and technical status notes.

## LLM memory

`docs/llm_memory/` contains the local LLM memory benchmark notes and the HPM-v2 / LLM adapter roadmap.

## Repo/process

`docs/repo/` contains repository cleanup notes, README design notes, migration notes, and roadmap material.

## Archive

`docs/archive/` contains older planning notes retained for provenance.

## Run logs

`docs/run_logs/` contains historical run logs.
