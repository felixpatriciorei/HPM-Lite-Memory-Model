# Figure reference notes

The figure script follows these research-figure standards:

- Show raw points where sample size is small.
- State what error bars mean.
- Use vector formats for final figure editing.
- Use readable sans-serif fonts and clear panel letters.
- Use colorblind-conscious colors and avoid red/green-only encoding.
- Avoid interpreting bookkeeping columns as real metrics.
